Installing Fedora Workstation 23


I was able to get a picture of all the components of the server.
In odrer to get the the server working with an OS. I had to boot from a bootable USB that had Fedora Workstation.
Thanks to Universal USB Installer.
Make sure you can boot from the USB first in the Bios.
The server was having issues with the normal boot.
Therefore, I went with
Troubleshooting
Choose the first option.
Start Fedora Live in basic graphics mode.
Once it boots, you have the option to try Fedora or install it.
Choose:
Install to Hard Drive
The setup begins.
Choose your language.
Click on:
Continue
I didn't do much on this screen.
I decided to go with:
Automatic partitioning selected
for the installation destination.
Click on:
Begin Installation
While it is installing, I made a root password by clicking:
root password
But I did not create a user. I suggest on doing this now else you will have to do it later like I had to.
Once it installs, the screen will ask you to create a user.
I used
csci211
for Full Name and User Name.
Enter your password too.
Click on:
Done
That's all! Fedora is ready to go!
