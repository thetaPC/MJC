Connecting to the Linode server remotely


Configure the bios to be able to allow console redirection. Go to the advanced tab in the bios, change the console redirection to match the picture.
Create the:
/etc/init/ttyS0.conf
just like the picture shows onto the local computer.
Connect the serial cable from the server to the local computer. The local computer should have the usb end.
Install the following tools on the local computer:
setserial
screen
By typing:
sudo apt-get install setserial
sudo apt-get install screen
You can verify that the cable is being read with two options.
1. Type:
dmesg | grep 'ttyUSB'
You should see it listed.
2. Type
ls -l /dev/ttyU*
Again, it should pop up.
Once you have verified that it's reading correctly, change the permission of it.
Type:
sudo chmod 777 /dev/ttyUSB0
In my case, mine is called USB0.
Type:
sudo start ttyS0
This won't work if you don't have the ttyS0.conf
And you only have to start it once.
Type:
sudo screen /dev/ttyUSB0 115200
Important: If you don't type sudo before screen then the tool will be wonky with funky symbols.
It takes a few minutes for the screen tool to work. Once it starts up, you will be able to see everything that the server is displaying.
To see if there are other screens up, type:
screen -r
To exit screen, enter
CTRL+A+:
kill
Configure grub in:
/etc/default/grub
By adding the commands that the picture has displayed to the left.
