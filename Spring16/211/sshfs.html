Using sshfs


We will be using sshfs to mount a remote to a local. Install sshfs if you do not have it:
                sudo apt-get install sshfs
                Installing will prompt you for the root password. Notice
                that my local will be maria1994 on my Ubuntu box. The remote for this example will be
                mloza1@jmkll.org
            

        

        
        

            
            

                Create a directory called mnt in the local and remote. Notice that there are no files in
                the mnt directory in local.
            

        

        
        

            
            

                Whereas, the remote has two files already created before this tutorial. 
            

        

        
        

            
            

                
In your local, type the following into your command line: 
        sshfs userName@server:/pathToYourDesiredMntPoint ~/pathToYourLocalMntPoint
       		userName@server is the information of your remote. Followed by the mountpoint 
       		in your remote finishing off with the mountpoint in your local. You will be 
       		prompted for the remote's password. To verify that you successfully mounted, typing: 
       		df 
       		into the command line should reveal that the remote is under the list.

            

        

        
        

            
            

                Local will now have files under the mnt directory. Keep in mind that these files are saved
                in the remote and are NOT saved under local. They will disappear after unmounting.
            

        

        
        

            
            

                Creating a new file in the mounted local in the mnt directory, will indeed be saved in the 
                remote. Once more, the new file will not be saved in local.
            

        

        
        

            
            

                Luckily, we can copy files from remote to the local. After unmounting, the copied file will 
                still appear in the location you wished to save it at in local.
            

        

        
        

            
            

                
Lastly, unmounting. Type: 
                fusermount -u /home/localUser/mnt  
                
                Make sure you are no longer in the local mnt directory!!! And that is how you mount local to remote using sshfs.
