How to add disks and create a logical volume


Open up Vmware. Double click on Hard drive. You will more than likely have only one drive. I've already added hard drives to mine.
Click on
Add...
at the bottom.
Select:
Hard Drive
After this, if you encounter a screen that I don't mention than select the default.
Change
Minimal disk size
to:
2 GB
Type:
fdisk -l | less
This will let you see what drives are there.
SYou should have seomething like this after typing: fdisk -l | less
I will make partitions on /dev/sdb and /dev/sdc and /dev/sdd. However, I will only show how to partition one since it's the same process.
Type:
fdisk /dev/sdx (Replace x with your appropriate letter)
Type for each:
n
p
1
Press Enter followed by another Enter
t
8e (This selects LVM partition type)
p
w
Now we'll start creating physical volumes (PV), volume groups (VG), and logical volumes (LG). The order is important from what I understand. I will only show this for one partition. Plus I will also show how to add another pv to a vg that's already been connected to an existing lv.
Type:
pvcreate /dev/sdb1 (This creates a physical volume using the partitioned drive)
Type:
vgcreate name-of-vg /dev/sdx (Creates a volume group with whatever name you want to call your vg and adds the drive to this vg)
lv create -L 2G -n name-of-lv name-of-vg (Creates a logical volume with whatever name you want and adds the vg to the lv)
mkdir /mnt/name-of-directory (call your new mounted directory whatever you want)
mkfs -t ext4 /dev/name-of-vg/name-of-lv (Makes filesystem type ext4)
mount /dev/name-of-vg/name-of-lv /mnt/name-of-directory
Type:
df
You should now see your newly mounted directories.
Type:
pvdisplay (This shows all the physical volumes)
vgdisplay (This shows all the volume groups)
lvdisplay (This shows all the logical volumes)
Type:
cat /etc/fstab (You should see the contents of fstab
Type:
vi /etc/fstab
From my understanding, we will edit the fstab file to automatically mount /dev/name-of-vg/name-of-lv to /mnt/name-of-directory everytime the OS boots.
Add at the following two lines to the bottom of the file:
/dev/mapper/name-of-vg-name-of-lv /mnt/name-of-directory1 ext4 rw,noatime 0 0
/dev/mapper/name-of-vg-name-of-lv /mnt/name-of-directory2 ext4 rw,noatime 0 0
If you want to add a partitioned drive to a vg that has been added to a lv then we must extend vg and lv to make room.
Type:
vgextend name-of-vg /dev/sdx lvextend -Lsize-you-want-to-addG /dev/name-of-vg/name-of-lv (This will automatically put the vg in the extended lv)
