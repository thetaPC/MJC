3ware raid


I moved the hard drives onto another server to test if the server will boot with Fedora. I made sure that the hard drives were in the same exact spot as they were in my original server.
I started the new server. It booted exactly the same way. The boot sequence and the raid (successful message) was the same. The only differene was the graphics since the new server had a different bios.
From my understanding, the raid configuration is only stored into the hard drive. Therefore, it doesn't matter what computer you install them into. The computer will boot from the hard drive with the hard drive's raid.
In order to create a different raid to fit the need of the system, first you have to delete the existing raid. Instead of choosing:
Create Unit
in the bios, select:
Delete Unit
Once you have deleted the unit, you may now create a new raid. Follow these instructions to create a new raid.
